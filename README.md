# 🎓 RFID-Based Entry & Access System  
## GCETTB Reunion – 100 Years Celebration  

This project is an RFID-based entry and access management system developed for the GCETTB 100-year reunion celebration. It is built to streamline attendee verification, enhance security, and ensure a smooth and contactless entry experience.

## 🚀 Features  
- 📡 RFID-based identification for attendees  
- ⚡ Fast and contactless entry system  
- 🔐 Secure access control  
- 🧾 Real-time logging of entries and exits  
- 💻 Integration with database for attendee management  
- 🔄 Scalable for large-scale events  

## 🛠️ Tech Stack  
- Microcontroller ( ESP8266 / ESP32)  
- RFID Module (RC522)  
- Backend (Node.js – Express.js)  
- Database (Mongo DB)  

## 🎯 Objective  
To modernize and automate the entry system for the GCETTB centenary reunion by reducing manual effort, avoiding crowd congestion, and maintaining accurate attendance records.
